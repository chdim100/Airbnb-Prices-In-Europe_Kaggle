# Airbnb-Prices-In-Europe_Kaggle
A Kaggle Project that can be found in the following link:
https://www.kaggle.com/datasets/thedevastator/airbnb-prices-in-european-cities

The current solution uses Python with Pandas, MatplotLib, Seaborn and LinearRegression from sklearn. The project's purpose is to inspect the Airbnb dataset, 
considering the factors that affect the rental fees in 10 European Cities during weekdays and weekends.
